SelandaTV


¿Qué es SelandaTv?

Se trata de un reproductor de canales de televisión de todo el mundo que emitan en abierto.


¿Qué tecnología utiliza?

Desarrollado en Angular, utiliza una petición dinámica de canales a una lista de GitHub para reproducirlos en el reproductor Videogular


Declaración de responsabilidad

SelandaTV ha sido desarrollado con fines educativos y sin ánimo de lucro, el creador no se hace responsable del contenido que se reproduzca ni de el uso que se le dé.
